Inicio de las prácticas

Objetivos

Inicializarme con el uso del GitHub pages y la plataforma RoboticsAcademy

Durante el inicio de las prácticas, el primer paso que realicé fue seguir la guía oficial de usuario de Robotics Academy, disponible en el siguiente enlace: https://jderobot.github.io/RoboticsAcademy/user_guide/ .

Una vez revisada la documentación, he estado aprendiendo y experimentando con GitHub Pages, entendiendo cómo funciona y mejorando el aspecto del primer intento

Pasos realizados:

El primer paso es clonar el repositrio donde se creará el GitHub pages. Una vez clonado creo dentro de la carpeta docs, otras dos subcarpetas (posts e images) y dos archivos (index.html y style.css). En la capeta "posts", servirá para añadir cada una de las semanas a la página, por otro lado, en la carpeta "images", se guardara las imagenes para añadir a la página. El archivo "index.html" define el contenido y la estructura de la página principal del bloque, mientras que el archivo "style.css", define el diseño visual de la página.

Despues de varias pruebas los archivos "index.html" y "style.css" quedan de esta forma:
