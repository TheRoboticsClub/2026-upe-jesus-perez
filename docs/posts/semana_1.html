Objetivos

Inicializarme con el uso de GitHub Pages y la plataforma Robotics Academy.

Durante el inicio de las prácticas, el primer paso que realicé fue seguir la guía oficial de usuario de Robotics Academy, disponible en el siguiente enlace: https://jderobot.github.io/RoboticsAcademy/user_guide/ .

Una vez revisada la documentación, he estado aprendiendo y experimentando con GitHub Pages, entendiendo cómo funciona y mejorando el aspecto del primer intento.

Pasos realizados

El primer paso fue clonar el repositorio donde se crearía el GitHub Pages. Una vez clonado, creé dentro de la carpeta docs dos subcarpetas (posts e images) y dos archivos (index.html y style.css).

La carpeta posts sirve para añadir cada una de las semanas al blog, mientras que la carpeta images almacena las imágenes utilizadas. El archivo index.html define la estructura y el contenido principal, y style.css define el diseño visual.

Después de varias pruebas, los archivos index.html y style.css quedaron de la siguiente forma:
