Inicialización con GitHub Pages

Semana 1 ·

Objetivos

Inicializarme con el uso de GitHub Pages y la plataforma Robotics Academy.

Durante el inicio de las prácticas, el primer paso que realicé fue seguir la guía de usuario de Robotics Academy, disponible en: https://jderobot.github.io/RoboticsAcademy/user_guide/ .

Tras revisar la documentación, estuve aprendiendo y experimentando con GitHub Pages, comprendiendo su funcionamiento y mejorando el aspecto del primer intento.

Pasos realizados

Se clonó el repositorio y se creó la carpeta docs con las subcarpetas posts e images, junto a los archivos index.html y style.css.

El archivo index.html define la estructura del blog y style.css el diseño visual.
